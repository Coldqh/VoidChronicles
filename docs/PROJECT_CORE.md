# VOID CHRONICLES — PROJECT CORE v0.1

## Genre
Procedural space-exploration roguelike with historical simulation, tactical expeditions and irreversible consequences.

## Player role
The player commands one active ship and a captain. The captain participates in landings, can be permanently injured or killed, and later versions can continue through companions, heirs or the ship AI.

## Core fantasy
Choose an unknown signal, risk the ship and expedition, recover evidence, survive the return and decide who receives the discovery.

## Core loop
Seed → galaxy generation → route selection → fuel expenditure → system scan → planet choice → cellular expedition → combat or hazard → discovery → extraction → archive/cargo → repair, sale and next jump.

## Three main systems
1. Incomplete scans and uncertain threats.
2. Cellular expeditions with persistent injuries and loss.
3. Causal world history: civilizations create events, ruins and artifacts.

## Deliberate exclusions in v0.1
No fleet management, dynasties, detailed diplomacy, romance, deep economy, colony management or full crew simulation.

## MVP success criteria
The full loop works on desktop and mobile, save data survives reload, same seed creates the same initial galaxy, and every ruin/artifact references an originating civilization.
