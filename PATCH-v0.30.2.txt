Void Chronicles v0.30.2 — Expedition Location Variation Hotfix

Исправлено:
- Разные локации снова имеют разное число объектов.
- Меняется только количество вторичных улик и свидетельств.
- Количество обязательных объектов сюжетной задачи не меняется.
- Генерация остаётся детерминированной по seed и ID локации.
- Повторные посещения сохраняют прежнюю карту.
- Версия приложения поднята до 0.30.2.

Причина:
После Lorebound Expeditions число объектов определялось только requiredEvidence.
Для однотипных миссий оно всегда было одинаковым, из-за чего падал
locationMemory.test.ts.

Установка:
1. Распаковать архив в C:\VoidChronicles.
2. Выполнить: node APPLY-v0.30.2.mjs
3. Выполнить: npm run check
